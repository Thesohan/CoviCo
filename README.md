# covico
CoviCo keeps the track of number of affected people in India due to Corona virus.

Concepts/Tools Used:
1- Bloc architecture
2- Dependency Injection
3- RxDart
4- Charts
5- Provider
6- Dio

https://api.covid19india.org/data.json
Above api is used to get the latest data.
